User Guide
==========

Practical guidance for implementing OpenSTEF in your projects — from common forecasting scenarios to production deployment and data integration.

**Use Cases** (:doc:`use_cases`)
   Practical examples for common forecasting scenarios: congestion management, transport forecasts, grid losses, and more.

**Deployment** (:doc:`deployment`)
   Production deployment patterns covering simple scheduled scripts, containerized pipelines, and enterprise integration approaches.

**Data Integration** (:doc:`data_integration`)
   Connect OpenSTEF to your data sources — S3, Databricks, InfluxDB, PostgreSQL, or custom backends.

**Migration from V3 to V4** (:doc:`migration_v3_v4`)
   Breaking changes, updated APIs, and step-by-step instructions for upgrading existing V3 implementations to V4.
