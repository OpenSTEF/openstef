OpenSTEF 4.0.0 documentation#

OpenSTEF is a comprehensive library for creating short term forecasts for the energy sector. It contains all components for the machine learning pipeline required to make a forecast.

Install#

pip install openstef

uv add openstef

conda install -c conda-forge openstef

pixi add openstef

For more detailed instructions, see the installation guide.
